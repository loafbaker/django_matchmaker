django_matchmaker
=================

A typical recommendation system built by django framework

Run in Django 5.2 and Python 3.10, with support from Bootstrap 3 and sqlite3.

## Updated version in 2026

- Upgrade to Django 5.2
- Migrate from Python 2 to Python 3
- Add `uv` support
- Add unit test for the app `matches`

## Run the unit test

```bash
python manage.py test matches
```
